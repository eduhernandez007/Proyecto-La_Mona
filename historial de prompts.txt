Soy estudiante de Ingeniería Eléctrica en la Universidad de Chile. Tengo un proyecto final para el ramo EL-4203 Programación Avanzada. El proyecto se llama "Sistema La Mona" y consiste en una plataforma web para gestionar las Olimpiadas Interdepartamentales CDI.

Tenemos solo pocos días para mostrar algo funcional. Somos 2-3 personas con conocimiento de Python y bases de datos, pero poco conocimiento del dominio.

Necesito que construyas la estructura base del proyecto con:

Backend en Python con FastAPI y SQLAlchemy (SQLite como base de datos)
Modelos OOP para: Departamento, Jugador (con género y estrellas), Equipo (básquetbol) y Partido
Una validación de negocio real usando clase abstracta: máximo 5 estrellas por equipo, mínimo 5 jugadores, mínimo 2 por género
Endpoints REST para crear/listar cada entidad, y registrar resultados de partidos
Un frontend HTML en una sola página que consuma el backend con fetch()
Dockerfile y docker-compose para poder desplegarlo
Inicializar git y hacer el primer commit con formato feat(scope): descripción
El foco principal está en demostrar OOP correctamente (herencia, abstracción, encapsulamiento).
